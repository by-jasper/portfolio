# HR Operations & Payroll Portfolio

A clean, recruiter-friendly personal portfolio and HR knowledge portal for a Singapore-based HR Operations and Payroll professional. The site highlights HR Shared Services, payroll, compliance, HR systems, process improvement, automation, and responsible AI use in HR.

## Tech Stack

- Static HTML
- CSS embedded in `index.html`
- Google Fonts / Inter
- No backend, API keys, or build framework required

## How to Install

No package installation is required for the current static site.

If you want a simple local server, use Python:

```bash
python3 -m http.server 8000
```

## How to Run Locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## How to Build

No build step is required. The site is a single static HTML file and can be deployed as-is to any static hosting provider.

## Where to Update Content

Update portfolio copy, sections, case studies, skill groups, AI statements, HR portal categories, and contact blocks in:

```text
index.html
```

Search for `TODO` comments in `index.html` before publishing.

## Where to Replace Resume

The resume buttons point to:

```text
resume.pdf
```

Place the final resume PDF at the repository root using that filename, or update the links in `index.html` to match the final file path.

## Where to Update Contact Details

Update the placeholder email address and LinkedIn URL in the Contact section and footer of `index.html`.

Current placeholders to replace:

- `hello@example.com`
- `#` LinkedIn links

## Privacy and Data Protection Reminder

Do not publish confidential HR data, employee names, salary details, NRIC / FIN / passport numbers, medical details, disciplinary matters, internal documents, vendor-sensitive details, or internal system screenshots. Use only dummy, anonymised, or generic content for examples.
