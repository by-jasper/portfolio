Google Sheets update guide

If you later connect this chatbot to Google Sheets, keep the same tree columns so HR can update content without touching code.

  1. Create a sheet with columns: nodeId, parentId, buttonLabel, message, and sortOrder.
  2. Each topic row uses welcome as parentId. Each sub-topic row uses its topic nodeId as parentId.
  3. Publish the sheet as CSV or expose it through Google Apps Script as JSON.
  4. Replace the local chatTree object with a small fetch loader that builds the same parent/child structure.
  5. Review privacy before publishing: do not store confidential employee cases or personal data in a public sheet.
